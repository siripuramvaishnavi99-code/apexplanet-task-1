# Basics of Web Development

## 📌 Project Overview

This project is a beginner-friendly web development project created to demonstrate the fundamentals of HTML, CSS, and JavaScript.

The webpage includes structured content, custom styling, and basic interactivity to provide a complete introduction to front-end web development.

---

## 🚀 Features

### HTML
- Semantic webpage structure
- Headings and paragraphs
- Images
- External links

### CSS
- Modern and responsive design
- Custom colors and typography
- Card-based layout
- Hover effects
- Proper spacing and alignment

### JavaScript
- Interactive button
- Alert message on click event

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
